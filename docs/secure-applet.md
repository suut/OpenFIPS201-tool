# `./openfips201.py secure-applet`

This command sets the applet to the `SECURED` state which prevents further usage of administrator commands
ran through the GlobalPlatform secure channel.

## Usage

```
usage: openfips201.py secure-applet [-h]

options:
  -h, --help  show this help message and exit
```

## Example

```sh
./openfips201.py secure-applet
```
